# NovelReaderTemplate

刚学Android的时候写的电子书应用，最初的版本是为一个女生写的魔兽世界的小说阅读器，后来废物利用，干起了盗版小说的勾当，由于只替换小说内容和应用包名就可以新生成换一本小说，又作了自动化生成工具，进入了批量生产时代。后来接到Google的警告函，导致帐号被封，里面的钱也未能取出来。算是白干一场。
